# Simulador Examen de Evaluación de Desempeño 2026

Abre `index.html` directamente o súbelo a Netlify/GitHub Pages.

Incluye:
- 40 preguntas convertidas a texto por OCR desde el PDF nuevo.
- Temporizador de 60 minutos.
- Modo oscuro.
- Corrección inmediata.
- Explicación al seleccionar.
- Resultados y revisión final.

Nota: el PDF original estaba compuesto por fotos/capturas, por lo que el texto fue extraído con OCR. Revisa `questions.js` para corregir cualquier palabra mal reconocida.
