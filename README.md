# Examen de Evaluación de Desempeño 2026

Simulador responsivo basado en el PDF nuevo. Incluye pantalla de inicio, temporizador de 60 minutos, modo oscuro, navegación por preguntas, corrección inmediata, explicación, resultados y revisión.

## Uso local
```bash
npm install
npm run dev
```

## Deploy en Netlify
- Build command: `npm run build`
- Publish directory: `dist`

## Editar preguntas
Las preguntas y claves están en `src/questions.js`.
Las imágenes del PDF están en `public/pages`.
